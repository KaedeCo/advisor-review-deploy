## 一、执行摘要

### 1.1 调研背景

本研究旨在为开发一款**具有自我搜索能力的导师评价查询应用**提供全面的技术可行性论证。该应用采用前后端分离架构，使用免费搜索引擎 DDGS (DuckDuckGo) 作为核心搜索能力，目标是帮助研究生/准博士生在选择导师时获取多维度、客观的评价信息。

### 1.2 核心发现

| 发现项 | 结论 |
|--------|------|
| **总体可行性** | ✅ **完全可行**，建议分梯队实施（共5大梯队）|
| **最佳数据源组合** | gradchoice.org + daoshipingjia.net + **小木虫** + 保研论坛 + 百度贴吧 + LetPub + GitHub开源数据 |
| **核心技术挑战** | 验证码处理(知乎)、签名算法逆向(小红书)、致谢词提取(CNKI) |
| **预计开发周期** | 核心框架 2 周，完整系统 6-8 周（含第五梯队 +3-5天）|
| **预期数据覆盖** | 20万+ 导师，70万+ 条评价，覆盖 1000+ 所院校 |
| **新增梯队价值** | 🆕 第五梯队提供"选导过程实录"和"联系导师时间线"，与结构化评价形成完美互补 |

### 1.3 关键结论

1. **第一梯队的 gradchoice.org** 是性价比最高的切入点（评分 8.0/10）
2. **百度贴吧是社交平台中最易爬取的**（评分 7.5/10），纯HTML无反爬
3. **知乎数据质量最高但反爬最强**（评分 4.5/10），需投入逆向成本
4. **🆕 小木虫 + 保研论坛 = 第五梯队王牌组合**（均评分 7.5/10），Discuz!架构爬取零门槛
5. **🆕 保研论坛拥有全网独有的"联系导师全过程实录"数据**（邮件回复时间线、面试题风格）
6. **第五梯队的核心价值在于"过程性数据"** 与第一梯队"结果性评分"形成完美互补
4. **小红书不推荐起步阶段尝试**（评分 3.0/10），反爬达商业级别
5. **CNKI 致谢词价值极高但获取困难**（评分 3.0/10），建议走机构库替代路径
6. **GitHub 开源数据集应作为预加载基线**，零成本启动

### 2.3 自动化评分算法思路

```python
# 伪代码 - 多维度加权评分
def calculate_advisor_score(reviews: list[dict]) -> dict:
    """
    输入: 来自多个源的评价列表
    输出: 六维得分 + 综合得分 + 置信度
    """
    scores = {}
    
    for dim in DIMENSIONS:
        # 收集该维度的所有信号
        signals = extract_dimension_signals(reviews, dim)
        
        # 加权融合（来源权重 × 时间衰减 × 可信度权重）
        weighted_scores = []
        for signal in signals:
            w = SOURCE_WEIGHTS[signal.source] * TIME_DECAY(signal.date) * signal.reliability
            weighted_scores.append(w * signal.value)
        
        # 归一化到 1-10 分
        scores[dim] = normalize(sum(weighted_scores), 1, 10)
    
    # 综合得分（可配置权重）
    scores['overall'] = sum(scores[dim] * dim.weight for dim in DIMENSIONS)
    
    # 置信度（基于样本量）
    scores['confidence'] = min(len(reviews) / 10, 1.0)
    
    return scores
```

#### 爬取方案

```python
import requests
import json
from bs4 import BeautifulSoup
import time
import random

class GradChoiceScraper:
    """研选 GradChoice 爬虫"""
    
    BASE_URL = "https://gradchoice.org"
    
    def __init__(self):
        self.session = requests.Session()
        self.session.headers.update({
            'User-Agent': 'Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36',
            'Accept': 'application/json, text/html',
            'Referer': self.BASE_URL,
            'Accept-Language': 'zh-CN,zh;q=0.9'
        })
        self.token = None
    
    def _get_token(self):
        """从首页或Cookie中提取API Token"""
        if not self.token:
            resp = self.session.get(self.BASE_URL)
            soup = BeautifulSoup(resp.text, 'html.parser')
            # 尝试从script标签或meta标签提取token
            scripts = soup.find_all('script')
            for script in scripts:
                if script.string and 'token' in script.string.lower():
                    import re
                    match = re.search(r'token["\']?\s*[:=]\s*["\']([^"\']+)', script.string)
                    if match:
                        self.token = match.group(1)
                        break
            # 备选：从cookie获取
            if not self.token:
                self.token = self.session.cookies.get('gc_token', '')
        return self.token
    
    def search_advisor(self, name: str, university: str = "") -> list[dict]:
        """搜索导师"""
        token = self._get_token()
        
        # 方案1：直接调用内部API（需抓包确认实际端点）
        # api_url = f"{self.BASE_URL}/api/advisors/search"
        # params = {'q': name, 'university': university, 'token': token}
        
        # 方案2：通过HTML页面解析
        url = f"{self.BASE_URL}/search"
        params = {'keyword': name}
        if university:
            params['school'] = university
        
        resp = self.session.get(url, params=params)
        resp.encoding = 'utf-8'
        soup = BeautifulSoup(resp.text, 'html.parser')
        
        results = []
        # 解析结果卡片（需根据实际HTML结构调整选择器）
        cards = soup.select('.advisor-card') or soup.select('.result-item')
        for card in cards:
            try:
                result = {
                    'name': card.select_one('.name').get_text(strip=True),
                    'university': card.select_one('.school').get_text(strip=True),
                    'department': card.select_one('.dept').get_text(strip=True) if card.select_one('.dept') else '',
                    'rating': float(card.select_one('.score').get_text(strip=True)) if card.select_one('.score') else None,
                    'review_count': int(card.select_one('.count').get_text(strip=True)) if card.select_one('.count') else 0,
                    'detail_url': card.select_one('a')['href'],
                    'source': 'gradchoice.org'
                }
                results.append(result)
            except Exception as e:
                continue
        
        # 限速
        time.sleep(random.uniform(1, 3))
        return results
    
    def get_reviews(self, advisor_url: str) -> list[dict]:
        """获取导师详情页的所有评价"""
        url = f"{self.BASE_URL}{advisor_url}" if not advisor_url.startswith('http') else advisor_url
        resp = self.session.get(url)
        soup = BeautifulSoup(resp.text, 'html.parser')
        
        reviews = []
        review_items = soup.select('.review-item') or soup.select('.comment')
        for item in review_items:
            try:
                review = {
                    'author': item.select_one('.author').get_text(strip=True)[:20],  # 截断防隐私
                    'rating': self._parse_rating(item.select_one('.stars')),
                    'date': item.select_one('.date').get_text(strip=True),
                    'content': item.select_one('.content').get_text(strip=True),
                    'dimensions': self._parse_dimensions(item),
                    'helpful': int(item.select_one('.helpful-count').get_text()) if item.select_one('.helpful-count') else 0
                }
                reviews.append(review)
            except:
                continue
        
        return reviews
    
    def _parse_rating(self, stars_el):
        """解析星级评分"""
        if not stars_el:
            return None
        cls = stars_el.get('class', [])
        for c in cls:
            if 'star-' in c or 'rating-' in c:
                return float(c.split('-')[-1])
        return None
    
    def _parse_dimensions(self, item):
        """解析多维评分"""
        dims = {}
        dim_items = item.select('.dimension-score')
        for d in dim_items:
            label = d.select_one('.dim-label').get_text(strip=True)
            score = d.select_one('.dim-value').get_text(strip=True)
            dims[label] = score
        return dims


# 使用示例
if __name__ == '__main__':
    scraper = GradChoiceScraper()
    
    # 搜索某位导师
    results = scraper.search_advisor("张三", "清华大学")
    print(f"找到 {len(results)} 条结果")
    
    for r in results[:3]:
        print(f"\n导师: {r['name']} | {r['university']} | 评分: {r['rating']}")
        reviews = scraper.get_reviews(r['detail_url'])
        print(f"评价数: {len(reviews)}")
```

```python
    def __init__(self):
        self.session = requests.Session()
        self.session.headers.update({
            'User-Agent': 'Mozilla/5.0 (Windows NT 10.0; Win64; x64) '
                          'AppleWebKit/537.36 (KHTML, like Gecko) '
                          'Chrome/120.0.0.0 Safari/537.36',

            'Accept-Language': 'zh-CN,zh;q=0.9,en;q=0.8',
        })
        self.request_count = 0
        self.captcha_triggered = False
        self.last_request_time = 0
        self.MIN_INTERVAL = 3  # 最小请求间隔（秒）
    
    def _rate_limit(self):
        """智能限速"""
        now = time.time()
        elapsed = now - self.last_request_time
        if elapsed < self.MIN_INTERVAL:
            time.sleep(self.MIN_INTERVAL - elapsed + random.uniform(0, 2))
        self.last_request_time = time.time()
        self.request_count += 1
    
    def _detect_captcha(self, response: requests.Response) -> bool:
        """检测是否触发验证码"""
        indicators = ['captcha', 'verify', 'geetest', '滑块', '验证']
        text = response.text.lower()
        return any(ind in text for ind in indicators)
    
    def _handle_captcha(self) -> bool:
        """处理验证码"""
        self.captcha_triggered = True
        print(f"[!] 第 {self.request_count} 次请求后触发验证码")
        
        # 策略1：等待冷却（通常5-10分钟）
        print("[*] 等待600秒冷却...")
        time.sleep(600)
        
        # 策略2：更换Session Cookie
        self.session.cookies.clear()
        self.session.get(self.BASE_URL)
        
        # 策略3：如果需要自动识别，集成ddddocr
        # from dddocr import DdddOcr
        # ocr = DdddOcr(show_ad=False)
        # captcha_img = ... # 获取验证码图片
        # code = ocr.classification(captcha_img)
        
        self.captcha_triggered = False
        self.request_count = 0
        return True
    
    def search_advisor(self, name: str, school: str = "") -> list[dict]:
        """搜索导师（带限速和验证码检测）"""
        self._rate_limit()
        
        params = {'keyword': name}
        if school:
            params['school'] = school
        
        try:
            resp = self.session.get(
                f"{self.BASE_URL}/search",
                params=params,
                timeout=15
            )
            
            if self._detect_captcha(resp):
                self._handle_captcha()
                return []  # 冷却后需重试
            
            resp.encoding = 'utf-8'
            soup = BeautifulSoup(resp.text, 'html.parser')
            
            results = []
            # 选择器需根据实际页面调整
            items = soup.select('.advisor-item') or soup.select('.list-item') or soup.select('tr')
            
            for item in items:
                try:
                    info = {
                        'name': self._safe_extract(item, ['.name', 'td:nth-child(1)']),
                        'school': self._safe_extract(item, ['.school', 'td:nth-child(2)']),
                        'department': self._safe_extract(item, ['.dept', 'td:nth-child(3)']),
                        'overall_score': self._extract_score(item),
                        'review_count': self._extract_count(item),

                    }
                    if info['name']:

                except:
                    continue
            
            return results
            
        except requests.exceptions.RequestException as e:
            print(f"[ERROR] 请求失败: {e}")
            return []
    
    def _safe_extract(self, element, selectors):
        """安全提取文本"""
        for sel in selectors:
            el = element.select_one(sel)
            if el:
                return el.get_text(strip=True)
        return ''

        if score_el:
            txt = score_el.get_text(strip=True)
            import re
            nums = re.findall(r'[\d.]+', txt)
            return float(nums[0]) if nums else None
        return None
```

```python
import requests
from bs4 import BeautifulSoup
import time
import random
import re

class TiebaScraper:
    """百度贴吧导师评价爬虫 - 第二梯队最友好平台"""
    
    SEARCH_URL = "http://tieba.baidu.com/f/search/res"
    
    def __init__(self):
        self.session = requests.Session()
        self.session.headers.update({
            'User-Agent': 'Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36',
            'Accept': 'text/html,application/xhtml+xml',
            'Accept-Language': 'zh-CN,zh;q=0.9',
            'Referer': 'http://tieba.baidu.com/',
        })
    
    def search_mention(self, advisor_name: str, university: str = "", 
                       keywords: list[str] = None) -> list[dict]:
        """
        在贴吧搜索导师相关讨论
        :param keywords: 扩展关键词，如 ["评价", "避坑", "体验", "怎么样", "导师"]
        """
        results = []
        
        base_query = advisor_name
        if university:
            base_query += f" {university}"
        
        # 组合多种搜索关键词
        queries = [base_query]
        if keywords:
            for kw in keywords:
                queries.append(f"{base_query} {kw}")
        
        for query in queries[:5]:  # 最多5种查询组合
            params = {'q': query}
            
            try:
                resp = self.session.get(self.SEARCH_URL, params=params, timeout=10)
                resp.encoding = 'gb2312'  # 贴吧编码
                
                soup = BeautifulSoup(resp.text, 'html.parser')
                
                # 解析搜索结果
                posts = soup.select('.s_post') or soup.select('.t')
                for post in posts:
                    try:
                        title_el = post.select_one('a.title') or post.select_one('a')
                        if not title_el:
                            continue
                        
                        result = {
                            'title': title_el.get_text(strip=True),
                            'url': f"http://tieba.baidu.com{title_el['href']}" if not title_el['href'].startswith('http') else title_el['href'],
                            'snippet': post.select_one('.p_content').get_text(strip=True) if post.select_one('.p_content') else '',
                            'forum_name': '',  # 所属贴吧
                            'query_used': query,
                            'source': 'tieba.baidu.com'
                        }
                        
                        # 去重
                        if not any(r['url'] == result['url'] for r in results):
                            results.append(result)
                    
                    except Exception as e:
                        continue
                
                time.sleep(random.uniform(1, 2))
                
            except Exception as e:
                print(f"[Tieba Error] {e}: query={query}")
                continue
        
        return results
    
    def get_post_detail(self, post_url: str) -> dict:
        """获取帖子详情和回复"""
        resp = self.session.get(post_url, timeout=10)
        resp.encoding = 'utf-8'
        soup = BeautifulSoup(resp.text, 'html.parser')
        
        # 提取楼主正文
        main_post = soup.select_one('#post_content_0') or soup.select_one('.d_post_content')
        content = main_post.get_text(strip=True) if main_post else ''
        
        # 提取所有回复
        replies = []
        reply_items = soup.select('.d_post_content') or soup.select('.j_d_post_content')
        for reply in reply_items[1:]:  # 跳过楼主
            replies.append({
                'content': reply.get_text(strip=True)[:500],
                'floor': reply.find_parent().select_one('.l_floor') if reply.find_parent() else ''
            })
        
        return {
            'main_content': content,
            'reply_count': len(replies),
            'replies': replies[:20],  # 只返回前20条回复
            'source_url': post_url
        }
```

```python
import requests
from bs4 import BeautifulSoup
import time
import random
import pandas as pd
from datetime import datetime
from typing import Optional

class LetPubScraper:
    """LetPub NSFC基金数据爬虫 - 推断导师科研实力"""
    
    BASE_URL = "http://www.letpub.com.cn"
    SEARCH_URL = f"{BASE_URL}/nsfcGrantSearch.php"
    
    def __init__(self):
        self.session = requests.Session()
        self.session.headers.update({
            'User-Agent': 'Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36',
            'Accept': 'text/html,application/xhtml+xml',
            'Referer': self.BASE_URL,
        })
    
    def search_by_pi(self, pi_name: str) -> pd.DataFrame:
        """按项目负责人（导师姓名）搜索NSFC基金项目"""
        all_grants = []
        page = 1
        
        while True:
            params = {
                'searchType': 'PI',
                'piName': pi_name,
                'page': page
            }
            
            try:
                resp = self.session.get(self.SEARCH_URL, params=params, timeout=15)
                resp.encoding = 'gb2312'
                soup = BeautifulSoup(resp.text, 'html.parser')
                
                # 解析基金列表表格
                table = soup.select_one('table') or soup.select_one('.grant-list')
                if not table:
                    break
                    
                rows = table.select('tr')[1:]  # 跳过表头
                if not rows:
                    break
                
                for row in rows:
                    cols = row.select('td')
                    if len(cols) >= 6:
                        grant = {
                            'project_code': cols[0].get_text(strip=True),
                            'project_name': cols[1].get_text(strip=True),
                            'pi_name': cols[2].get_text(strip=True) if len(cols) > 2 else pi_name,
                            'org_name': cols[3].get_text(strip=True) if len(cols) > 3 else '',
                            'grant_type': cols[4].get_text(strip=True) if len(cols) > 4 else '',
                            'amount_yuan': self._parse_amount(cols[5].get_text(strip=True)) if len(cols) > 5 else 0,
                            'start_year': self._parse_year(cols[6].get_text(strip=True)) if len(cols) > 6 else None,
                            'status': cols[7].get_text(strip=True) if len(cols) > 7 else '',
                            'source': 'letpub.com.cn',
                            'fetched_at': datetime.now().isoformat()
                        }
                        all_grants.append(grant)
                
                # 检查是否有下一页
                next_page = soup.select_one('a:contains("下页")') or soup.select_one('.next')
                if not next_page or page >= 10:  # 最多10页
                    break
                
                page += 1
                time.sleep(random.uniform(2, 5))
                
            except Exception as e:
                print(f"[LetPub Error] Page {page}: {e}")
                break
        
        df = pd.DataFrame(all_grants)
        return self._analyze_funding(df)
    
    def _parse_amount(self, amount_str: str) -> float:
        """解析金额字符串"""
        import re
        numbers = re.findall(r'[\d.]+', amount_str)
        return float(numbers[0]) if numbers else 0.0
    
    def _parse_year(self, year_str: str) -> Optional[int]:
        """解析年份"""
        import re
        years = re.findall(r'\d{4}', year_str)
        return int(years[0]) if years else None
    
    def _analyze_funding(self, df: pd.DataFrame) -> pd.DataFrame:
        """添加分析字段"""
        if df.empty:
            return df
        
        # 计算累计金额
        df['total_amount'] = df.groupby('pi_name')['amount_yuan'].transform('sum')
        
        # 判断项目级别权重
        type_weights = {
            '重大': 5, '重点': 4, '面上': 3, 
            '青年': 2.5, '地区': 1.5, '联合': 2
        }
        df['type_weight'] = df['grant_type'].map(type_weights).fillna(1)
        
        # 加权得分
        df['funding_score'] = df['amount_yuan'] * df['type_weight']
        
        return df.sort_values('start_year', ascending=False)
```

#### 致谢词NLP分析框架

```python
import jieba
from snownlp import SnowNLP
import re

class AcknowledgmentAnalyzer:
    """学位论文致谢词分析器 - 判断导师人品和指导风格"""
    
    POSITIVE_PATTERNS = [
        r"悉心.*指导", r"耐心.*教导", r"倾囊相授", r"如沐春风",
        r"严格要求.*关怀", r"不仅.*更.*", r"终身难忘",
        r"感激.*恩师", r"有幸.*师从", r"获益匪浅"
    ]
    
    NEGATIVE_PATTERNS = [
        r"基本.*不管", r"放养", r"联系不上", r"延期",
        r"自己想办法", r"几乎.*无指导", r"压力过大"
    ]
    
    RED_FLAGS = [
        r"换导师", r"退学", r"抑郁", r"崩溃", r"多年.*未毕业",
        r"后悔", r"噩梦", r"终于.*熬"
    ]
    
    @staticmethod
    def analyze(text: str) -> dict:
        """
        分析致谢词文本
        返回: 情感得分(-1~1), 关键信号列表, 风险等级
        """
        # 基础情感分析
        s = SnowNLP(text)
        sentiment_score = s.sentiments  # 0~1, 转换为-1~1
        adjusted_score = (sentiment_score - 0.5) * 2
        
        # 匹配特定模式
        positive_hits = []
        for pattern in AcknowledgmentAnalyzer.POSITIVE_PATTERNS:
            matches = re.findall(pattern, text)
            if matches:
                positive_hits.extend(matches)
        
        negative_hits = []
        for pattern in AcknowledgmentAnalyzer.NEGATIVE_PATTERNS:
            matches = re.findall(pattern, text)
            if matches:
                negative_hits.extend(matches)
        
        red_flag_hits = []
        for pattern in AcknowledgmentAnalyzer.RED_FLAGS:
            matches = re.findall(pattern, text)
            if matches:
                red_flag_hits.extend(matches)
        
        # 计算风险等级
        risk_level = "LOW"
        if red_flag_hits:
            risk_level = "CRITICAL"
        elif len(negative_hits) > 2 and adjusted_score < -0.3:
            risk_level = "HIGH"
        elif negative_hits:
            risk_level = "MEDIUM"
        
        return {
            'sentiment_score': round(adjusted_score, 3),
            'positive_signals': positive_hits[:5],
            'negative_signals': negative_hits[:5],
            'red_flags': red_flag_hits,
            'risk_level': risk_level,
            'word_count': len(text),

        words = jieba.posseg.cut(text)
        names = [word for word, flag in words if flag == 'nr' and len(word) >= 2]
        return list(set(names))[:5]
```

```python
            'Accept': 'text/html,application/xhtml+xml',
            'Accept-Language': 'zh-CN,zh;q=0.9',

                       max_pages: int = 5) -> list[dict]:
        """

        :param keyword: 搜索关键词（如导师姓名）

                'mod': 'forumdisplay',
                'searchsubmit': 'yes',
                'searchtext': keyword,
            }
            if board_id:
                params['fid'] = board_id
            
            try:

                    f"{self.BASE_URL}/forum.php",
                    params=params,
                    timeout=15
                )
                resp.encoding = 'gbk'  # DZ常用编码
                
                soup = BeautifulSoup(resp.text, 'html.parser')
                
                # 解析帖子列表（DZ标准选择器）

                              soup.select('.sptable tr')
                
                if not thread_items:
                    break  # 无更多结果
                
                for item in thread_items:
                    try:
                        title_el = item.select_one('a.s.xst') or item.select_one('th a')
                        if not title_el or not title_el.get('href'):
                            continue
                        
                        # 提取tid
                        href = title_el['href']
                        tid_match = re.search(r'tid=(\d+)', href)
                        tid = tid_match.group(1) if tid_match else ''
                        
                        result = {
                            'title': title_el.get_text(strip=True),
                            'tid': tid,
                            'url': f"{self.BASE_URL}/{href}" if not href.startswith('http') else href,
                            'author': '',
                            'reply_count': 0,
                            'view_count': 0,
                            'last_post_time': '',
                            'source': 'muchong.com',
                            'source_tier': 'tier5_academic'
                        }
                        
                        # 提取作者

                                   item.select_one('.by cite a')
                        if author_el:
                            result['author'] = author_el.get_text(strip=True)

                                  item.select_one('.num em')
```

```python
                'orderby': 'lastpost',

                        })
                    except:

    def extract_contact_timeline(self, post_url: str) -> dict:
        """
        从保研经验帖中提取"联系导师时间线"
        这是保研论坛独有的高价值数据
        """
        resp = self.session.get(post_url, timeout=15)
        soup = BeautifulSoup(resp.text, 'html.parser')
        
        content_div = soup.select_one('#post_0 .t_f') or soup.select_one('#post_0 td.t_f')
        text = content_div.get_text() if content_div else ''
        
        timeline = []
        
        # 正则匹配日期模式 + 动作关键词
        date_pattern = r'(\d{1,2}[月/]\d{1,2}日?|\d{1,2}[/\.]\d{1,2}|\d{4}[-/]\d{1,2}[-/]\d{1,2})'
        action_keywords = ['发', '邮件', '回复', '收到', '面试', 'offer', '录取', '拒绝', '接受']
        
        sentences = re.split(r'[。\n]', text)
        for sentence in sentences:
            has_date = re.search(date_pattern, sentence)
            has_action = any(kw in sentence for kw in action_keywords)
            if has_date and has_action:
                timeline.append({
                    'date': has_date.group(1),
                    'content': sentence.strip(),
                    'type': self._classify_action(sentence)
                })
        
        return {
            'timeline': timeline,
            'full_text_length': len(text),
            'contact_events_count': len(timeline),
            'inferred_response_speed': self._infer_response_speed(timeline)
        }
    
    @staticmethod
    def _classify_action(text: str) -> str:
        if any(w in text for w in ['发', '发送', '投']):
            return 'sent_email'
        elif any(w in text for w in ['回复', '收到回复', '回信']):
            return 'received_reply'
        elif any(w in text for w in ['面试', '见面', '复试']):
            return 'interview'
        elif any(w in text for w in ['offer', '录取', '接收']):
            return 'admitted'
        elif any(w in text for w in ['拒绝', '没过', '失败']):
            return 'rejected'
        return 'other'
    
    @staticmethod
    def _infer_response_speed(timeline: list[dict]) -> str:
        if len(timeline) < 2:
            return 'unknown'
        # 计算首次发邮件到首次回复的时间间隔
        sent_idx = next((i for i, t in enumerate(timeline) if t['type'] == 'sent_email'), None)
        reply_idx = next((i for i, t in enumerate(timeline) if t['type'] == 'received_reply' and i > (sent_idx or 0)), None)
        if sent_idx is not None and reply_idx is not None:
            return 'fast' if (reply_idx - sent_idx) <= 2 else ('normal' if (reply_idx - sent_idx) <= 4 else 'slow')
        return 'unknown'
```

## 十一、DDGS 搜索策略详解

### 11.1 核心搜索模式

```python
from duckduckgo_search import DDGS
import asyncio
import concurrent.futures
from typing import Optional
from dataclasses import dataclass
from enum import Enum

class SourceTier(Enum):
    TIER1_PRO = "professional_platform"
    TIER2_SOCIAL = "social_community"
    TIER3_ACADEMIC = "academic_auxiliary"

@dataclass
class SearchResult:
    """标准化搜索结果"""
    title: str
    url: str
    snippet: str
    source: str
    tier: SourceTier
    relevance_score: float  # 0-1
    found_at: str  # 哪个网站命中

class DDGSSearchEngine:
    """基于DDGS的多源智能搜索引擎"""
    
    # 各梯队的站点配置
    TIER1_SITES = [
        ("gradchoice.org", 1.0),
        ("daoshipingjia.net", 0.95),
        ("daoshikoubei.net", 0.75),
        ("pi-review.com", 0.65),
        ("ratemysupervisor.net", 0.85),
        ("daoshipingjia.org", 0.45),
    ]
    
    TIER2_SITES = [
        ("zhihu.com", 0.90),
        ("xiaohongshu.com", 0.55),
        ("tieba.baidu.com", 0.95),
        ("1point3acres.com", 0.88),
        ("douban.com", 0.72),
    ]
    
    TIER2_KEYWORDS = ["评价", "避坑", "体验", "吐槽", "怎么样", "读研", "研究生"]
    
    def __init__(self, max_results_per_site: int = 10):
        self.ddgs = DDGS()
        self.max_results = max_results_per_site
        self.cache = {}  # 生产环境用Redis
    
    async def search_advisor(self, 
                            advisor_name: str, 
                            university: str = "",
                            tiers: list[SourceTier] = None) -> list[SearchResult]:
        """
        多源并发搜索导师评价
        :param tiers: 指定搜索梯队，None表示全部
        """
        if tiers is None:
            tiers = [SourceTier.TIER1_PRO, SourceTier.TIER2_SOCIAL]
        
        tasks = []
        
        if SourceTier.TIER1_PRO in tiers:
            tasks.extend([
                self._search_tier1(advisor_name, university, site, weight)
                for site, weight in self.TIER1_SITES
                if weight >= 0.7  # 只搜索高权重站点
            ])
        
        if SourceTier.TIER2_SOCIAL in tiers:
            tasks.extend([
                self._search_tier2(advisor_name, university, site, weight, kw)
                for site, weight in self.TIER2_SITES
                if weight >= 0.7
                for kw in self.TIER2_KEYWORDS[:3]  # 每站只用前3个关键词
            ])
        
        # 并发执行所有搜索任务
        results = await asyncio.gather(*tasks, return_exceptions=True)
        
        # 合并、去重、排序
        flattened = [r for r in results if isinstance(r, SearchResult)]
        deduplicated = self._deduplicate(flattened)
        sorted_results = self._rank_results(deduplicated)
        
        return sorted_results[:50]  # 返回Top50
    
    async def _search_tier1(self, name: str, univ: str, site: str, weight: float) -> list[SearchResult]:
        """第一梯队：site: 限定搜索专业平台"""
        query = f'"{name}" site:{site}'
        if univ:
            query = f'"{name}" "{univ}" site:{site}'
        
        try:
            results = self.ddgs.text(query, max_results=self.max_results)
            return [
                SearchResult(
                    title=r['title'],
                    url=r['href'],
                    snippet=r.get('body', ''),
                    source=site,
                    tier=SourceTier.TIER1_PRO,
                    relevance_score=weight,
                    found_at=site
                ) for r in results
            ]
        except Exception as e:
            print(f"[DDGS Error] {site}: {e}")
            return []
    
    async def _search_tier2(self, name: str, univ: str, site: str, 
                           weight: float, keyword: str) -> list[SearchResult]:
        """第二梯队：site: + 关键词扩展搜索社交平台"""
        base = f'"{name}" {keyword}'
        if univ:
            base = f'"{name}" "{univ}" {keyword}'
        query = f'{base} site:{site}'
        
        try:
            results = self.ddgs.text(query, max_results=self.max_results // 2)
            return [
                SearchResult(
                    title=r['title'],
                    url=r['href'],
                    snippet=r.get('body', ''),
                    source=site,
                    tier=SourceTier.TIER2_SOCIAL,
                    relevance_score=weight * 0.8,  # 社交平台略降权
                    found_at=f'{site}[{keyword}]'
                ) for r in results
            ]
        except Exception as e:
            print(f"[DDGS Error] {site}/{keyword}: {e}")
            return []
    
    def _deduplicate(self, results: list[SearchResult]) -> list[SearchResult]:
        """URL去重"""
        seen_urls = set()
        unique = []
        for r in results:
            if r.url not in seen_urls:
                seen_urls.add(r.url)
                unique.append(r)
        return unique
    
    def _rank_results(self, results: list[SearchResult]) -> list[SearchResult]:
        """多因子排序"""
        for r in results:
            # 综合相关性、来源权重、标题匹配度
            name_match = 1.0  # 简化：实际应计算名称匹配分数
            final_score = r.relevance_score * name_match
            r.relevance_score = final_score
        
        return sorted(results, key=lambda x: x.relevance_score, reverse=True)


# 使用示例
async def main():

        print()

if __name__ == '__main__':
    asyncio.run(main())
```

### 11.3 搜索结果融合算法

```python
def merge_and_rank(raw_results: list[SearchResult], 
                  local_index: dict) -> list[dict]:
    """
    融合DDGS实时搜索结果与本地索引数据
    """
    scored_results = []
    
    for result in raw_results:
        # 基础分：来源权重
        base_score = result.relevance_score
        
        # 加分项：
        bonus = 0.0
        
        # 1. 本地索引命中加分
        if result.found_at in local_index:
            bonus += 0.2
            result.has_local_data = True
        
        # 2. 标题包含完整姓名加分
        if advisor_name_lower in result.title.lower():
            bonus += 0.15
        
        # 3. snippet包含评价关键词加分
        eval_keywords = ['评价', '避坑', '体验', '推荐', '不建议']
        if any(kw in result.snippet for kw in eval_keywords):
            bonus += 0.1
        
        # 4. 专业平台额外加分
        if result.tier == SourceTier.TIER1_PRO:
            bonus += 0.1
        
        # 最终得分（带上限）
        final_score = min(base_score + bonus, 1.0)
        
        scored_results.append({
            **result.__dict__,
            'final_score': final_score
        })
    
    # 按最终得分降序排列
    return sorted(scored_results, key=lambda x: x['final_score'], reverse=True)
```
